# Getting Started with Agentic Workflows

_Learn how to set up and use GitHub Agentic Workflows to create AI-powered agents in your repository._

## Welcome

- **Who is this for**: Developers, DevOps engineers, and anyone interested in AI-powered automation with GitHub Copilot.
- **What you'll learn**: How agentic workflows are structured, how to create custom AI agents, and how to integrate them into your development workflow.
- **What you'll build**: A custom AI agent that works with GitHub Copilot in your repository.
- **Prerequisites**:
  - A GitHub account
  - Basic familiarity with GitHub repositories, branches, and pull requests
  - Access to GitHub Copilot

- **How long**: This exercise takes less than 15 minutes to complete.

In this exercise, you will:

1. Explore an agentic workflow setup and open a pull request
1. Create a custom AI agent definition file
1. Merge your changes to activate the agent

Simply copy the exercise to your account, then give your favorite Octocat (Mona) **about 20 seconds** to prepare the first lesson, then **refresh the page**.
