# Getting Started with Agentic Workflows

<img src="https://octodex.github.com/images/Professortocat_v2.png" align="right" height="200px" />

Hey arilivigni!

Mona needs help managing her new Astro-powered **GitHub Info** website. The site lives in this repository, and Mona wants to use **agentic workflows** to keep it current with updates from her own notes, the **GitHub Blog**, and the **GitHub Changelog**.

In this GitHub Skills exercise, you'll practice how to:

- install the repository setup for GitHub Agentic Workflows
- create an agentic workflow that drafts website updates in a pull request
- operationalize that workflow so Mona can run it manually and on a schedule
- document how Mona reviews automated changes before they reach `main`

Helpful starter files in this repository:

- `site/content/github-info.md`
- `notes/mona-notes.md`

This exercise is self-paced, so feel free to take a break whenever you need one. ☕️
